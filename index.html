Experience 'Feed the Bunny,' a charming interactive game developed using JavaScript and CSS Animation. In this game, set in a serene and scenic environment, adorable bunnies playfully emerge from behind bushes. Your mission? Feed as many bunnies as possible within just 20 seconds. Your success is tracked by the game, adding an element of competition to the experience.

This website "Banana Binge" employs Node.js, Express, and RESTful APIs to craft a dynamic hub for TV enthusiasts. The platform adeptly assembles the top 10 trending TV shows, showcasing comprehensive descriptions and captivating trailers in a unified space. Fueled by the Movie Database API, the site delivers real-time updates on popular TV shows, while the YouTube API seamlessly integrates compelling trailers.

"SerenityNow" is a web application developed using JavaScript and the Chart.js library, designed to help users track and manage daily anxiety, meditation, and caffeine consumption. It offers a user-friendly interface for inputting data and generates insightful graphs, showcasing progress over time. Additionally, it incorporates the Zen Quote API to provide a daily inspirational quote, promoting mindfulness.

Opulente is an online store created using Node.js, ExpressJS, MongoDB, and Pug. This combination makes shopping easy, allowing you to browse products and make purchases smoothly. Pug makes the website look great, while MongoDB manages data efficiently.
